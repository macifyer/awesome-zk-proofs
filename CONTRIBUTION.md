# Contribution Guide

Thanks for contributing!

## Add new links
- All new resources must be added to `src/data/graphData.ts`.
- Do not edit the awesome list directly in `README.md`.

## Regenerate the awesome list
```bash
npm run awesome
```

This updates the section between `<!-- AWESOME_LIST:START -->` and
`<!-- AWESOME_LIST:END -->` in `README.md`.
